# MLOps
This repo contains the basic structure required to start MLOps projects

create python virtual environment
```bash
python -m venv ~/.venv
```
activate environment
```bash
source ~/.venv/bin/activate
```
install libraries
```bash
make install
```
lint project
```bash
make lint
```
test project
```bash
make test
```
